# Fish Match 3

Небольшая браузерная игра "3 в ряд" с разноцветными рыбками.

## Публикация на GitHub Pages

1. Создай новый публичный репозиторий на GitHub.
2. Загрузить в него нужно файл `index.html`.
3. В настройках репозитория включи GitHub Pages:
   `Settings -> Pages -> Build and deployment -> Source -> Deploy from a branch`.
4. Выбери ветку `main` и папку `/ (root)`.
5. После публикации сайт будет доступен по ссылке вида:
   `https://USERNAME.github.io/REPOSITORY/`
